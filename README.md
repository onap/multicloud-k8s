# MultiCloud-k8-plugin

MultiCloud Kubernetes plugin for ONAP multicloud.

# Installation

Requirements:
* Go 1.10
* Dep

Steps:

* Clone repo in GOPATH src:
    * `cd $GOPATH/src && git clone https://git.onap.org/multicloud/k8s`

* Run unit tests:
    *  `make build`

* Compile to build Binary:
    * `make deploy`
